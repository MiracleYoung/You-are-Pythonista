## Analysis

```python
str1 = input()
str2 = input()

def out(str):
    a = len(str) % 8
    if a != 0:
        str = str + ('0' * (8 - a))
    for i in range(len(str) // 8):
        print(str[i * 8:8 * (i + 1)])

out(str1)
out(str2)
```
