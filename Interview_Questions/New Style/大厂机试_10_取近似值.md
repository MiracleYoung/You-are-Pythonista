## Question
写出一个程序，接受一个正浮点数值，输出该数值的近似整数值。如果小数点后数值大于等于5,向上取整；
小于5，则向下取整。

```python
try:
    while True:
        a = eval(input())
        if (a - ____(a)) >= 0.5:
            print(int(a) ____ 1)
        else:
            print(int(a))
except:
    pass
```
