## Question

def xxx(x):
    result = [0,1]
    for index in range(x-2):

    return result

## Analysis

这是可指定长度的斐波那契数列

```python
def fibs(x):
    result = [0,1]
    for index in range(x-2):
        result.append(result[-1]+result[-2])
    return result

num = int(input('Enter a number'))
print(fibs(num))
```
