# 大厂机试_34_字符串最后一个单词的长度

## Question
计算字符串最后一个单词的长度，单词以空格隔开，下列代码正确吗（）？

```python
a = input().split()
if len(a) > 1:
    print(len(a[-1]))
else:
    print(len(a[0]))
``` 

%!A. 正确!%

%!B. 不正确!%

----

## Answer
@!A!@

----

## Analysis

思路：split() 空格为界，字符串转列表

```python
a = input().split()
if len(a) > 1:
    print(len(a[-1]))
else:
    print(len(a[0]))
```
