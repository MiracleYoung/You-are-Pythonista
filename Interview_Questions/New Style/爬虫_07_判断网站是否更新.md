# 爬虫_07_判断网站是否更新

## Question
怎么判断网站是否更新（）？

%!A. 比较网页 Length !%

%!B. 比较网页 Size !%

%!C. 使用 MD5 数字签名!%

----

## Answer
@!C!@

----

## Analysis

使用 MD5 数字签名。

每次下载网页时，把服务器返回的数据流 ResponseStream 先放在内存缓冲区，然后对 ResponseStream 生成 MD5 数字签名 S1，下次下载同样生成签名 S2，比较 S2 和 S1，如果相同，则页面没有更新，否则网页就有更新。
